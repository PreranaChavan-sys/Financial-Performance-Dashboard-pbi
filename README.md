# Financial Performance Dashboard (Power BI)

## 📊 Overview
This project is an interactive dashboard to analyze Sales, Cost, Profit, and Profit Margin.

## 🚀 Features
- KPI cards for Sales, Cost, Profit, Profit %
- Quarterly sales trend
- Region & product analysis
- Discount analysis
- Interactive slicers (Country, Product, Segment)

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization

## 💡 Insights
- Identified top-performing regions
- Analyzed profit vs sales
- Evaluated discount impact
